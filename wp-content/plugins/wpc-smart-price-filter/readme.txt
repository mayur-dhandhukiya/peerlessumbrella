WPC Smart Price Filter is an advanced widget help you to filter products by price with unlimited price ranges.

== Description ==

Running an online store with products of wide-range prices? Use the **WPC Smart Price Filter for WooCommerce** plugin to help your customers find just the right products that fit their budget at ease.

**WPC Smart Price Filter for WooCommerce** allows users to insert an advanced price filter widget that can be easily configured to be displayed at any place using the shortcode. Users can set up as many price ranges as they prefer to reflect the different levels of products in their store. Upon clicking on a price range, all appropriate products whose price falls on the chosen range will be filtered out and displayed for buyers to choose from.

= Features =

- Unlimited customizable price ranges
- Editable price label for each range
- Display the widget anywhere on your site using the shortcode
- Show/hide the product counts for each range
- Compatible with most common WordPress themes and plugins

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Smart Price Filter", Install & Activate it
4. After that, you can go to Appearance >> Widgets to add widget "WPC Smart Price Filter" to the sidebar.
