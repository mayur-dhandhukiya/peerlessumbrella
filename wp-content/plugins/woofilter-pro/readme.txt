To upgrade Product Filter by WBW plugin free version to the commercial you need to
1. Install latest free version of the plugin here: https://wordpress.org/plugins/woofilter

2. Then download commercial version from your personal account: https://woobewoo.com/my-account/

!Important! You need both versions to be installed. Free and commercial one.
