=== APPROVE NEW USERS REGISTRATION WORDPRESS & WOOCOMMERCE PLUGIN ===

Contributors: Codiffy

== Description ==

Approve new users registration for manually & allow specific users to login to your site. Admin can also activate or disable the users login access in bulk from users screen & set the custom message for login error. The plugin allows to send notifications for account apporoval or account disapproval for users.
