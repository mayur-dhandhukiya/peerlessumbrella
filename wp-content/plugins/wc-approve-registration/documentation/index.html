Approve New Users Registration WordPress & WooCommerce Plugin

Discription

Approve new users registration for manually & allow specific users to login to your site. Admin can also activate or disable the users login access in bulk from users screen & set the custom message for login error. The plugin allows to send notifications for account apporoval or account disapproval for users.

Documentation

Install & Activate Plugin

  1. Extract plugin folder from zip
  2. Upload plugin folder to wp-content/plugins/
  3. Activate plugin from Admin dashboard's menu "Plugins"

Initials Settings

After installing the plugin the plugin creates a new admin menu page "Approve Customer" with Submenus "Settings", "Prices Rule Sets", "Exchange Rates" each is defined below.

Enable plugin to function on frontend.


The "Enable Module" must be checked in order to function the plugin on frontend, if you want to temporary disable the plugin on frontend just uncheck this.

Enable Manual Approve: Enable this option to manually approve new customer registration.

Message For Account Created: Message for customers when account is created but pending for approval.

Account Disabled Login Message: Message for customers when account is disabled for login.

Notification Settings.


Enable Account Pending Notification : Enable this option to enable the account pending notification.

Account Pending Notification Subject : Email notification subject for account in pending.

Account Pending Notification Body : Account approval email notification body..

Enable Account Approve Notification : Enable this option to enable the account approve notification.

Account Approval Notification Subject : Email notification subject for account approval.

Account Approve Notification Body : Account approval email notification body.

Enable Account Disapprove Notification : Enable this option to enable the account disapprove notification.

Account Disapprove Notification Subject : Email notification subject for account disapprove.

Account Disapprove Notification Body : Account disapprove email notification body.
