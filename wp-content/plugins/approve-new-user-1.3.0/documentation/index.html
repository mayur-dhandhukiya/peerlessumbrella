1. How to Install

1: Install From Wordpress Admin:

  1. Download zip file from codecanyon.
  2. Logged into wordpress admin panel.
  3. From menu navigate to plugins
  4. Click on Add New.
  5. Click Upload Plugin and then upload zip file that you hav downloaded from codecanyon. After installation click on Activate plugin.

 

2:  Install via FTP:

  1. Unzip the zip file that you have downloaded from codecanyon into you computer.
  2. Upload files via any FTP cient like Filezilla.
  3. After uploading all files logged in to wordpress admin panel.
  4. From menu navigate to Plugin and then click on Installed Plugins.
  5. Click on install button under then Addify – Approve New User

General Settings

1) In General settings you can spesify on which users roles you want to apply approve new user, if you to apply on all you can select all.

2) If you want to show user roles drop down select on the registration page then you can use that module that is mention, this module will show user role selection drop down on registration page.

3) You can also add the messages, these messages are shown when user is registred, pending and when rejected.

 

 

 

 

 

Email Settings

Here you can email messages that are sent to user.

1) Approval Email Message:

When user is registred then an email is sent to user to tell him/her that his/her account is pending. You can send customize message to the user by using this email message.

2) Disapprove Email Message:

When account is disapprove by admin then an email is sent to user to tell him/her that his/her account is disapproved by admin. You can customize this message from here.

3) Approved Email Message:

When account is approved by admin then an email is sent to user to tell him/her that his/her account is approved. You can customize this email from here.

Additional Field Settings

If you want to get some additional information like (tax number or License no) then you can add this additional field on the registration form.
